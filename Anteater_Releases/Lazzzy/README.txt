-----------------
 개요
-----------------
Lazzzy는 스마트폰을 통해 원격 조작가능한 토글 스위치 입니다.

-----------------
 구성
-----------------
Lazzzy는 Arduino에서 작동하는 프로그램과 Android 환경에서 작동하는 어플리케이션으로 구성됩니다.
Arduino 회로도는 .\Lazzzy_arduino\Circuit 에서 확인할 수 있습니다.

-----------------
 사용
-----------------
git 저장소에서 소스 파일을 복사해 코드에서 다음 부분을 수정해야 합니다.
Arduino:
- 접속할 AP의 이름
- 접속할 AP의 비밀번호
- 포트 번호
Android:
- Arduino가 할당받을 IP 주소
- 포트 번호

Arduino에 전원을 연결하면 프로그램은 설정한 AP를 찾아 접속하는 과정을 거칩니다.
접속 과정 동안 LED에 불이 들어오고, 최대 약 30초 정도 소요됩니다.
Android의 어플리케이션에서 버튼을 누를 때 마다 Arduino의 서보 모터를 토글할 수 있습니다.
또한 Arduino에 연결된 토글 스위치를 통해 직접 서보 모터를 토글할 수 있습니다.

-----------------
 주의
-----------------
Arduino와 Android는 같은 네트워크 상에 존재해야 합니다.
두 소스 코드에서 수정한 포트 번호는 서로 같아야 합니다.
