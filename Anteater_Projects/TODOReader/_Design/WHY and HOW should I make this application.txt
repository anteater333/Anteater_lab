끝으로... HtmlAgilityPack으로 할지 아니면 HttpRequest로 할지 고민좀 해보자.

HtmlAgilityPack은 html 파싱용 라이브러리로, URL만 가져다 주면 그냥 http 리퀘스트 알아서 해서 해당 페이지의 html을 가져와준다. 몹시 쉽고, 그냥 쉽다 뭐 더 할말이 있나.

HttpWebRequest를 쓰는 방법은, 내가 직접 'GET' 메소드로 리퀘스트 해서 리스폰스 받고 처리하는 방법.
내가 이걸 왜 고민하느냐면, 자바스크립트 문법 배우다가 어쩌다보니 내가 이걸 하고있더라. http request.

배운거 익히는 식으로 그냥 이걸로 하자.
System.Net 네임스페이스를 사용하면 된다고 한다.
